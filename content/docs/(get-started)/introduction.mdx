## What is better-auth-invite-plugin?

better-auth-invite-plugin is a [Better Auth](https://www.better-auth.com) plugin that adds a complete invitation system to your app, allowing you to create and manage invites for both new users and existing accounts.
The plugin is maintained by <GithubUser user="0-Sandy" userName="Sandy" /> and is not officially affiliated with Better Auth.

## How invites work

The plugin allows you to invite users to your app. It supports two main invitation types:
- **Private Invitations**: This invitations is targeted to a specific email address, ensuring only the intended recipient can use the invitation.
- **Public Invitations**: Can be used by anyone that has the invite, it's ideal for role invitations (example: giving the role "discord-member" if you joined from the app's discord server)
